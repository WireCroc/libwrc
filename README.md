# Libwrc
**Packet capture library.**

## Installation

1. Run `make install`

## Usage

```c
#include <wrc/wrc.h>
```

## Example

```c
int main(void) {
    wrc cap;
    wc_iflist ifc = wrc_get_interfaces();
    
    wrc_default(&cap);

    int opts = wrc_setopts(&cap, ifc.ifc[1], PA_NULL, 0);
    if (opts != 0) {
        printf("setopts error\n");
        exit(0);
    }

    wrc_cap(&cap, WCFILE, NULL, DEFAULT_CAP);
    
    wrc_destroy(&cap);
    
    return 0;
}
```
